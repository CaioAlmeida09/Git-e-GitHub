Aprendendo a utilizar o New Terminal

ja conhecemos que a sequência para criar nossos repositórios git é:

  1. Inicializar o Repositório
  2. Changes - Fazer as mudanças necessárias
  3. Staged Changes - Confirmar pro servidor aquelas mudanças
  4. Commit - Fazer o print do código naquele momento
  5. Pus - Subir pro git

Passo a passo terminal jhkhj (new terminal)

  1. new repository (no prório git)
  2. Abrir o terminal (new terminal)
  3. Para a inclusão no repositório precisamos digitar: git init
  4. para o envio da alteração para o staged digitamos: git add .
  5. para o commit digitamos: git commit m"meu segundo commit terminal"
